yaf phpunit library for yaf application test.

Requirements
------------

**PHP5.4.0 or later**

Installation
------------

Create or modify your composer.json

``` json
{
    "require": {
        "lancerhe/php-yafunit": "dev-master"
    }
}
```

-----

See https://github.com/lancerhe/yaf-phpunit
